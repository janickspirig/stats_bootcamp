Retrieves rows where there is a match between tables A and B based on a common key.
```sql
SELECT *  
FROM A  
INNER JOIN B ON A.key = B.key;
```

Retrieves rows where there is a match in either table A, table B, or both — including all data from both tables.

```sql
SELECT *
FROM A
FULL JOIN B ON A.key = B.key;
```

Rows in A without a match in B will still appear, with `NULL` values for B’s columns.
```sql

FROM A  
LEFT JOIN B ON A.key = B.key;
```
