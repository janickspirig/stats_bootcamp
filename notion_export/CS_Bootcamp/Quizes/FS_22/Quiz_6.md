- Correct, inside `find_connection()` we call the Open Transport API and inside `find_restaurants()` we call the Yelp API, thus we have two HTTP requests. In both cases we retrieve data from the servers, thus we have two HTTP GET requests.
- Incorrect, with `obscure` we perform a slicing operation on the restaurants list in line 11, we can not perform slicing with a string, this would lead to an error
```python
liste = [1,2,3,4]
liste[:"janick"]

TypeError: slice indices must be integers or None or have an __index__ method
```
- Correct, `obscure` is an integer, so that we can filter out the first x restaurants in line 11
- Incorrect, when we call the Yelp API we want to retrieve information about available restaurants and not modify any resource on the server, thus we use a GET and not a POST request
