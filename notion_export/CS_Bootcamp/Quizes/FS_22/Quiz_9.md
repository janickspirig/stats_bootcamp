We see that in the column `products` we have the information stored about how many items of each product were bought. 
For each purchase (record in the DataFrame), we want to know now how many items were bought on average per product type.
In our example we have 9 different products and 12 items in total. Thus, on average `12 / 9 = 1.33` items were bought. 
How can we compute this now using Python?
To get the sum of product items in the purchase we can execute:
```python
ORDERS.loc[3, 'products'].values()
[3, 1, 2, 1, 1, 1, 1, 1, 1] # -> all values in the dictionary

sum(ORDERS.loc[3, 'products'].values())
12 # -> sum of products bought
```
To get the number of different products in the purchase we can execute:
```python
ORDERS.loc[3, 'products'].keys()
['book', 'lamp', 'smartphone', 'door', 'car', 'banana', 'chair', 'robot', 'wallet'] # -> names of all the different products bought

len(ORDERS.loc[3, 'products'].keys())
9 # -> number of different products bought
```
We can combine it all:
```python
sum(ORDERS.loc[3, 'products'].values()) / len(ORDERS.loc[3, 'products'].keys())
1.3333333333333333 # -> average number of items bought per product type in purchase
```
And finally to execute the calculation above on the entire DataFrame, i.e., for each purchase we can make use of the `.apply()` function: 
```python
df['products'].apply(lambda r : sum(r.values()) / len(r.keys()))
```
